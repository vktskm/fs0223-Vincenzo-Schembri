POSITION: STATIC

Il valore "static" è quello di default di position. Un elemento con position:static non è posizionato nella pagina in nessun modo "speciale", seguirà sempre il flow normale. Si dichiara esplicitamente quando NON volete che un elemento figlio erediti il valore del padre, oppure quando c'è la necessità di "resettare" esplicitamente un elemento in una determinata condizione.

Questo div è posizionato con position: static
